# AI Adaptive Quiz Platform

An AI-powered interactive quiz platform built during the SINTHANAI 2K26 24-Hour Hackathon.

## Features
- AI-generated quizzes
- Adaptive difficulty adjustment
- Multiplayer quiz competitions
- Leaderboards
- Progress analytics with charts

## Tech Stack
Frontend: Next.js  
Backend: Node.js  
AI: OpenAI API  
Database: Firebase  
Charts: Chart.js  
Deployment: Vercel

## Hackathon
SINTHANAI 2K26 – Erode Sengunthar Engineering College
